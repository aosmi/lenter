# Lenter Keyboard
### Optimized keyboard for low-end devices.
---
## Description: This keyboard was created for devices with limited capabilities.

## Functionality: Enter characters, edit, send.

## Key Highlights: Presence of Cyrillic and Latin alphabets, mathematical signs, flexible Enter button. Drawing on Canvas. Complex objects and functions were "cut out" as a victim of optimization and moved to a single file.

## Technical Characteristics:
#### Gradle: 8.0
#### Language: Java
#### DSL: Groovy DSL
#### API: 13 – 35 (Android 7.0 – Android 15)
