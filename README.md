# Lenter Keyboard
### Optimized keyboard for low-end devices.
---
## Description: This is my pet project with the aim of helping weak phones in the form of an optimized keyboard.

## Functionality: Enter characters, edit, send.

## Key Highlights: 
* Presence of Cyrillic and Latin alphabets.
* mathematical signs.
* flexible Enter button.
* Drawing on Canvas.
* Complex objects and functions were "cut out" as a victim of optimization and moved to a single file.
* "Break screen" function for phones with broken screen.

## Technical Characteristics:
#### Gradle: 8.0
#### Language: Java
#### DSL: Groovy DSL
#### API: 10 – 35 (Android 2.3.7 – Android 15)
